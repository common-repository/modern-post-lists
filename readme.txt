Modern Post Lists is a versatile WordPress plugin that offers customizable layouts for displaying posts, galleries, and portfolios in a stylish and engaging manner.

== Description ==

Modern Post Lists is an innovative WordPress plugin designed to enhance the way you display posts, galleries, and portfolios on your website. With its customizable layouts and user-friendly interface, this plugin makes it easy to create beautiful, responsive, and engaging content displays.

Key Features:
- Multiple layout options including Blog Posts, Full Width Gallery, Masonary Gallery, and Portfolio.
- Dynamic content sourcing from posts, pages, or attachments.
- Advanced category filtering for precise content display.
- Customizable call-to-action buttons with adjustable text settings.
- Integrated search bar for easy content navigation.
- Responsive design for a seamless experience across all devices.

Whether you are a blogger, photographer, artist, or business owner, Modern Post Lists offers the flexibility and tools needed to create stunning content presentations.

== Installation ==

1. Upload the 'modern-post-lists' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Modern Post Lists' settings page to configure the plugin.

For detailed instructions, please visit the plugin documentation page.

== Frequently Asked Questions ==

= Can I use Modern Post Lists with any theme? =

Yes, Modern Post Lists is designed to be compatible with most themes. However, some themes' specific layouts may require slight adjustments for optimal performance.

= Is the plugin responsive? =

Yes, all layouts are fully responsive and adapt to various screen sizes.
